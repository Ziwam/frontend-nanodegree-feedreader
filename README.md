## Setup
To get started, download the zip and open the **./index.html** file to view the feed application on your browser.
## Directions
Open the application's spec file **./jasmine/spec/feedreader.js** to view all tests.

Refer to the HTML (**./index.html**), CSS (**./css/style.css**) and JavaScript (**./js/app.js**) files to get in-depth understanding of the application's framework.
